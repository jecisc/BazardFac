Module Arbre_codage


module Arbre_codage: sig .. end
Module permettant de construire et dessiner un arbre de codage binaire.
Author(s): FIL - IEEA - Univ. Lille1 (oct 2010)

Le type


type 'a t
Le type des arbres de codages binaires.
exception Arbre_vide
Exception déclenchée par les fonctions Arbre_codage.racine_codante, Arbre_codage.racine_symbole, Arbre_codage.gauche et Arbre_codage.droit.

Fonctions de création et de représentation graphique


val creer : 'a array -> string array -> 'a t
creer alphabet code renvoie l'arbre du codage défini par l'alphabet et le code.
Raises Codage_incorrect si le code n'est pas binaire.
val dessiner : int t -> Pervasives.out_channel -> unit
dessiner a c imprime via le canal c une description de l'arbre binaire a au format Graphviz.

CU : le canal c doit être ouvert.
See also le site de Graphviz.


Fonctions primitives d'arbre


val racine_codante : 'a t -> bool
racine_codante a =
  • Vrai si la racine de l'arbre de codage a correspond à un symbole codé,
  • Faux sinon.

Raises Arbre_vide si l'arbre a est vide.
val racine_symbole : 'a t -> 'a
racine_symbole a renvoie le symbole codé par la racine de a.
Raises
  • Arbre_vide si l'arbre a est vide.
  • Poisition_non_codante si racine_codante a renvoie faux.
val gauche : 'a t -> 'a t
gauche a renvoie le sous-arbre gauche de l'arbre a.
Raises Arbre_vide si a est vide.
val droit : 'a t -> 'a t
droit a renvoie le sous-arbre droit de l'arbre a.
Raises Arbre_vide si a est vide.
val est_vide : 'a t -> bool
est_vide a =
  • Vrai si l'arbre a est vide,
  • Faux sinon.
