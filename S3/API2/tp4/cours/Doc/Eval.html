Module Eval


module Eval: sig .. end
Évaluateur d'expressions postfixées.

Utilise les modules Pile, Lexemes et Evaluation_postfixee.

Programme destiné à être compilé ocamlc -o eval pile.cmo lexemes.cmo evaluation_postfixee.cmo eval.ml

Usage : eval <expr> où <expr> = expression postfixée à évaluer.

Opérateurs autorisés :

  • + : addition
  • - : soustraction
  • * : multiplication
  • / : quotient entier
  • % : reste d'une division (modulo).

Author(s): FIL - IEEA - Univ. Lille1 (oct 2010, revu oct 2012)

val usage : unit -> 'a
