Module Lexemes


module Lexemes: sig .. end
Module d'analyse lexicale pour les expressions arithmétiques sur les entiers.
Author(s): FIL - IEEA - Univ. Lille1 (oct 2010)

Le type lexeme



type operateur =
| Add
| Sub
| Mul
| Div
| Mod

type lexeme =
| Par_ouvrante
| Par_fermante
| Int of int
| Op of operateur
Une expression arithmétique est constituée d'entiers, d'opérateurs et de parenthèses ouvrantes et fermantes. Un lexème est un entier ou un opérateur ou une parenthèse ouvrante ou fermante.
exception Caractere_non_autorise
Exception déclenchée par la fonction lexemes si une chaîne contient des caractères non autorisés.

L'analyseur lexical


val lexemes : string -> lexeme array
lexemes s renvoie un tableau de lexèmes dans l'ordre dans lequel ils apparaissent dans la chaîne s.

CU : s ne peut contenir que les caractères espace (' '), chiffres ('0' ...'9') et opérateurs ('+', '-', '*', '/', '%'). Déclenche l'exception Caractere_non_autorise sinon.
